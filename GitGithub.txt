Git-GitHub Notları

- git init ile klasörümüzde Git'i başlatırız.
- git add . komutu ile yaptığımız değişiklikleri working directory'den staging area'a taşırız.
- git status ile commit öncesinde nerede olduğumuzu görürüz.
- git commit -m "V1" komutu ile değişiklikleri staging area'dan commit store'a taşırız ve bir versiyon oluşturmuş oluruz.
- git diff ile working space deki değişiklikleri görebiliriz
- git show hashcode ilede commit store daki  değişiklikleri görürüz.
- git remote add origin url diyerek bir adres tanımlamış oluruz.
- git push -u origin master ilede dosyalarımızı push ederiz.
- git pull diyerek remote repository nin son halini alırız.

- git brach ile branc leri listeleriz.
- git branch --all ile remote deki brancleri de listeleriz.
- git branch name diyerek yeni bir branch oluştururuz
- git merge anotherBranch yazarsak brancleri birleştirmiş oluruz.
